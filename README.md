# Stream Deck 中国股票价格插件

实时显示中国股票、黄金、期货及大盘指数价格的 Stream Deck 插件。

## 功能特性

- 📈 **股票价格**：支持沪深主板、创业板、科创板
- 🪙 **黄金/期货**：国际金价、国内金价、银、铜等
- 📊 **大盘指数**：上证指数、深成指、创业板、沪深300等
- 🎨 **颜色提示**：
  - 上涨：红色背景 + 红色标题文字
  - 下跌：绿色背景 + 绿色标题文字
  - 平盘：灰色背景 + 灰色标题文字
- ⏱️ **自动刷新**：每5秒自动更新数据
- 📝 **中文支持**：股票名称中文显示
- ✂️ **自动截断**：超长名称自动截断至6字符
- 🔄 **点击切换**：点击按钮循环切换4种显示模式

## 显示模式

点击按钮可循环切换以下4种显示模式：

| 模式 | 第一行 | 第二行 | 第三行 | 说明 |
|------|--------|--------|--------|------|
| 0 | 名称 | 价格 | 涨跌幅% | 默认模式 |
| 1 | 名称 | H:最高价 | L:最低价 | 最高/最低价 |
| 2 | 名称 | V:成交量 | A:成交额 | 成交量/成交额 |
| 3 | 名称 | O:今开 | Y:昨收 | 今开/昨收价 |

**示例（贵州茅台）**：
```
模式0:              模式1:              模式2:              模式3:
贵州茅              贵州茅              贵州茅              贵州茅
¥1400.000          H:1426.00          V:300.87W          O:1400.00
+0.52%             L:1396.66          A:42.57E           Y:1401.18
```

背景颜色：红色=上涨，绿色=下跌，灰色=平盘

## 安装

### 方式一：直接安装（推荐）

1. 下载 `.streamDeckPlugin` 文件
2. 双击或在 Stream Deck 软件中导入
3. 重启 Stream Deck 软件

### 方式二：源码安装

```bash
# 克隆仓库
git clone https://github.com/zbr0310528/streamdeck.stockprice.git
cd streamdeck.stockprice/v2

# 安装依赖
npm install

# 构建
npm run build

# 打包
zip -r com.gdby.stockprice.streamDeckPlugin com.gdby.stockprice.sdPlugin/*

# 安装插件（Windows）
# 将 com.gdby.stockprice.streamDeckPlugin 复制到 %AppData%\Elgato\StreamDeck\Plugins\

# 重启 Stream Deck 软件
```

## 使用方法

1. 在 Stream Deck 按钮列表中找到「中国股票价格」
2. 拖动到需要的位置
3. 点击按钮，在弹出窗口中：
   - 选择类型（股票/大盘指数/黄金）
   - 输入代码
4. 点击保存
5. **点击按钮**：循环切换显示模式

## 数据来源

数据来自新浪财经 API。

## 系统要求

- Windows 10 及以上
- Stream Deck 软件 6.5 及以上
- Node.js 20（仅开发需要）
